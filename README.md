## 🎯 О проекте

ShuraOS PlayBook представляет собой автоматизированный инструмент настройки системы, использующий платформу **AME Wizard** (Automated Maintenance Engine) для применения комплексных оптимизаций Windows 11. Проект создан с целью предоставить пользователям готовую к использованию систему с оптимальными настройками производительности, приватности и удобства работы.

Плейбук применяет более 200+ настроек реестра, конфигураций служб и системных параметров, которые были тщательно отобраны и протестированы для обеспечения максимальной стабильности при значительном улучшении производительности системы.

## ✨ Ключевые особенности

### 🚀 Производительность
- **Оптимизация энергопотребления** — создание и активация высокопроизводительного плана электропитания с настройками для процессоров с P и E ядрами
- **Отключение ненужных служб** — деактивация более 30 системных служб, потребляющих ресурсы без пользы для обычного пользователя
- **Оптимизация памяти** — отключение сжатия памяти и оптимизация работы с оперативной памятью
- **Ускорение загрузки** — отключение гибернации и оптимизация процесса загрузки системы
- **Оптимизация PowerShell** — предварительная компиляция модулей для ускорения работы

### 🔒 Приватность
- **Полное отключение телеметрии** — блокировка сбора диагностических данных Microsoft
- **Отключение Windows Defender** (опционально) — возможность полного отключения встроенного антивируса
- **Блокировка сетевых соединений** — настройка файрвола для блокировки исходящих соединений телеметрии
- **Отключение рекламных идентификаторов** — блокировка персонализированной рекламы
- **Настройка облачных функций** — отключение синхронизации данных с облаком Microsoft

### 🧹 Очистка системы
- **Удаление Microsoft Edge** — полное удаление браузера Chromium от Microsoft
- **Удаление OneDrive** — полное удаление облачного клиента Microsoft
- **Удаление Teams** — деинсталляция корпоративного мессенджера
- **Удаление AI-компонентов** — удаление Copilot и функций Recall (ИИ-функции)
- **Удаление UWP-приложений** — очистка более 50 встроенных приложений Windows
- **Очистка временных файлов** — автоматическое удаление логов и временных файлов

### 🎨 Настройка интерфейса
- **Классическое контекстное меню** — возврат привычного меню правой кнопки мыши
- **Тёмная тема** — автоматическое применение тёмной темы системы
- **Кастомные обои** — установка обоев ShuraOS для рабочего стола и экрана блокировки
- **Настройка панели задач** — скрытие лишних значков и виджетов
- **Настройка проводника** — оптимизация функций и внешнего вида проводника
- **Настройка просмотрщика фотографий** — выбор между классическим просмотрщиком Windows и FlyPhotos

## 📦 Установка и использование

### Предварительные требования

Перед запуском плейбука убедитесь в следующем:

1. **Антивирус должен быть отключён** — ShuraOS PlayBook требует временного отключения защитника Windows
2. **Нет ожидающих обновлений** — не должно быть активных процессов обновления Windows
3. **Подключение к интернету** — необходимо для загрузки программного обеспечения
4. **Зарядка ноутбука** — система должна быть подключена к сети питания
5. **Администраторские права** — запуск от имени администратора обязателен

### Инструкция по установке

## Способ 1: Применение на уже установленной Windows (рекомендуется)

1. Скачайте и установите **[AME Wizard](https://ameliorated.io/)** с официального сайта.
2. Перейдите в релизы → [GitHub Releases — ShuraOS PlayBook](https://github.com/SanyaSavin/ShuraOS_PlayBook/releases)

   - **Если хотите проще и быстрее** — скачайте готовый файл `ShuraOS_PlayBook.apbx` из активов релиза → переходите сразу к шагу 5.
   - **Если хотите вручную (самая свежая версия)** — скачайте архив с исходниками (`Source code (zip)` или **Code → Download ZIP**).

3. **Только если выбрали вручную**:
   - Распакуйте архив в отдельную папку.
   - Выделите **всё содержимое** папки **кроме** папки `.git` (если она есть).
   - Правой кнопкой → **7-Zip** (или другой архиватор с шифрованием) → **Добавить к архиву…**
   - Параметры:
     - Имя файла: `ShuraOS_PlayBook.apbx` (расширение `.apbx` обязательно)
     - Формат: 7z
     - Пароль: `malte`
     - **Шифровать имена файлов** — обязательно включить
     - Метод шифрования: AES-256
   - Создайте архив.

4. Запустите **AME Wizard от имени администратора**.
5. Перетащите (drag & drop) файл `.apbx` (готовый из релиза или созданный вручную) в окно программы.
6. Настройте опции (если плейбук предлагает выбор, например, оставить Defender или нет).
7. Нажмите **Run** → дождитесь завершения (система перезагрузиться один раз, если не убрали галочку. От себя скажу, что лучше не убирайте галочку, перезагрузка все же необходима).

## Способ 2: Интеграция в ISO-образ Windows (для автоматической настройки при установке)

1. Скачайте свежий официальный ISO **Windows 11** (с сайта Microsoft).
2. Получите файл `.apbx` одним из двух способов:
   - **Проще** — скачайте готовый `ShuraOS_PlayBook.apbx` из [релизов](https://github.com/SanyaSavin/ShuraOS_PlayBook/releases).
   - **Вручную** — скачайте исходники и создайте `.apbx` точно так же, как описано выше в Способе 1 (шаги 3).
3. Запустите **AME Wizard от имени администратора**.
4. Перетащите **ISO-образ Windows** в окно (drag & drop).
5. Перетащите файл **ShuraOS_PlayBook.apbx**.
6. Выберите плейбук **ShuraOS** в списке.
7. Настройте параметры (если требуется).
8. Нажмите кнопку создания образа.

## 🔧 Функции и возможности

### Оптимизация производительности

#### Настройки энергопотребления

Плейбук создаёт специальный план электропитания «Ультра производительность» с оптимизированными настройками для современных процессоров Intel и AMD с гибридной архитектурой (P-ядра и E-ядра):

```yaml
# Конфигурация плана питания
PERFINCPOL: 2      # Минимальное состояние процессора
PERFDECPOL: 1      # Максимальное состояние процессора
PERFINCTHRESHOLD: 10  # Порог увеличения производительности
PERFDECTHRESHOLD: 8   # Порог уменьшения производительности
CPMINCORES: 100    # Минимальное количество активных ядер
```

**Преимущества:**
- Снижение задержек при отклике системы
- Оптимизация использования ресурсов для игр и продуктивной работы
- Улучшенное управление температурой процессора

#### Оптимизация памяти

```powershell
# Отключение сжатия памяти
Disable-MMAgent -mc
```

#### Отключение гибернации

**По умолчанию:** Отключено

```powershell
# Отключение гибернации
powercfg.exe /h off

# Настройка реестра
HiberbootEnabled: 0
```

**Преимущества:**
- Освобождение дискового пространства (файл hiberfil.sys может занимать до 8 ГБ)
- Устранение проблем при двойной загрузке
- Повышение стабильности при обновлении системы

**⚠️ Предупреждение:** Отключение гибернации может привести к перегреву ноутбуков в спящем режиме. На ноутбуках рекомендуется оставить гибернацию включённой.

### Приватность и безопасность

#### Отключение телеметрии

Плейбук полностью отключает сбор телеметрии Microsoft через множественные механизмы:

**Групповые политики:**
```yaml
AllowTelemetry: 0              # Запрет телеметрии
AllowCommercialDataPipeline: 0  # Запрет коммерческих данных
AllowDeviceNameInTelemetry: 0   # Запрет отправки имени устройства
DisableEnterpriseAuthProxy: 1   # Отключение прокси
MicrosoftEdgeDataOptIn: 0       # Запрет сбора данных Edge
AllowExperimentation: 0         # Запрет экспериментов
```

**WMI/Autologger:**
```yaml
Diagtrack-Listener Start: 0     # Отключить слушатель телеметрии
SQMLogger Start: 0              # Отключить логгер
SetupPlatformTel Start: 0        # Отключить телеметрию платформы
```

**Брандмауэр Windows:**
```
Block-Unified-Telemetry-Client  # Блокировка исходящих соединений
Block-Windows-Error-Reporting   # Блокировка отчётов об ошибках
```

#### Отключение Windows Defender

**⚠️ Внимание:** Отключение Defender значительно снижает безопасность системы.

Плейбук предоставляет возможность полного отключения Windows Defender:

```powershell
# Основные политики
DisableAntiSpyware: 1
DisableAntiVirus: 1

# Real-Time Protection
DisableRealtimeMonitoring: 1
DisableBehaviorMonitoring: 1
DisableOnAccessProtection: 1
DisableScanOnRealtimeEnable: 1
DisableIOAVProtection: 1

# Cloud Protection
SpynetReporting: 0
SubmitSamplesConsent: 2

# SmartScreen
EnableSmartScreen: 0
```

**Опции:**
- `disable-defender` — Отключить Защитник Windows (по умолчанию)
- `keep-defender` — Оставить Защитник Windows без изменений

# Настройки NVIDIA Telemetry
NVIDIA Corporation\NVControlPanel2\Client
OptInOrOutPreference: 0  # Отказ от телеметрии

## ⚠️ Известные ограничения

### Совместимость

1. **Windows 10** — Плейбук разработан специально для Windows 11 и не поддерживает Windows 10

2. **Windows 11 в режиме S** — Не работает из-за ограничений режима S

3. **Корпоративные версии** — Некоторые групповые политики могут конфликтовать с настройками плейбука

4. **BitLocker** — При использовании шифрования диска рекомендуется отключить BitLocker перед запуском плейбука


## ❓ Часто задаваемые вопросы

### Вопрос: Безопасен ли ShuraOS PlayBook?

**Ответ:** ShuraOS PlayBook использует проверенные методы оптимизации, применяемые сообществом уже много лет. Все изменения основаны на официальной документации Microsoft и лучших практиках сообщества. Однако любое изменение системы сопряжено с определёнными рисками. Рекомендуется создать точку восстановления системы и резервные копии важных данных перед запуском плейбука.

### Вопрос: Можно ли отменить изменения?

**Ответ:** Большинство изменений, внесённых плейбуком, могут быть отменены вручную через:
- Повторный запуск плейбука с противоположными настройками
- Ручное изменение параметров реестра
- Восстановление из точки восстановления системы

Однако некоторые операции являются необратимыми.

### Вопрос: Как часто следует запускать плейбук?

**Ответ:** Плейбук предназначен для однократного запуска после чистой установки Windows 11. Повторный запуск рекомендуется только после крупных обновлений Windows, которые могут сбросить некоторые настройки.

### Вопрос: Будет ли плейбук работать после обновления Windows?

**Ответ:** Крупные обновления Windows (например, 22H2 → 23H2) могут восстановить некоторые удалённые компоненты и сбросить настройки. Рекомендуется создать резервную копию конфигурации плейбука и повторно запустить его после обновления.

### Вопрос: Почему мой компьютер стал работать быстрее?

**Ответ:** ShuraOS PlayBook оптимизирует систему несколькими способами:
- Отключает более 30 системных служб, потребляющих ресурсы
- Оптимизирует планы электропитания для максимальной производительности
- Удаляет сотни мегабайт встроенных приложений, которые постоянно обновляются в фоновом режиме
- Отключает сжатие памяти и другие ресурсоёмкие функции
- Настраивает приоритеты процессов для лучшего отклика системы

### Вопрос: Влияет ли плейбук на игры?

**Ответ:** Да, положительно. Оптимизация системы включает:
- Настройку плана питания на максимальную производительность
- Отключение фоновых служб, потребляющих ресурсы CPU и RAM
- Оптимизацию сетевого стека для снижения задержек
- Отключение Game DVR и связанных функций
- Настройку приоритетов процессов

### Вопрос: Будет ли работать мой антивирус после отключения Defender?

**Ответ:** Если вы используете сторонний антивирус (Kaspersky, Avast, Bitdefender и др.), он продолжит работать корректно. Однако мы настоятельно рекомендуем иметь активную защиту в реальном времени для безопасности системы.

### Вопрос: Как сообщить о проблеме?

**Ответ:** Проблемы можно сообщить через [GitHub Issues](https://github.com/SanyaSavin/ShuraOS_PlayBook/issues). При сообщении о проблеме укажите:
- Версию Windows (Winver → About)
- Сборку Windows (Winver → Version)
- Шаги для воспроизведения проблемы
- Скриншоты ошибок (если применимо)
